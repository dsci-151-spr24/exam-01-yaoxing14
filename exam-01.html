    a. Calculate the proportion of flights that are delayed at departure at each of these time intervals.

    b. Comment on how the likelihood of being delayed change throughout the day? in the morning, will be more punctual. in the night will be delay.

Narrative: This analysis examines delayed flights at four time intervals throughout the day to understand variations in delays. Approach: I created hourly intervals, calculated the proportion of delayed flights (defined as departures delayed by more than zero minutes), and summarized the results for each time period.

library(tidyverse)
library(nycflights13)

# Analysis for delay proportions based on time intervals
flights_delay_analysis <- flights %>%
  mutate(
    time_interval = case_when(
      hour >= 0 & hour < 6  ~ "12:01 AM - 6:00 AM",
      hour >= 6 & hour < 12 ~ "6:01 AM - 12:00 PM",
      hour >= 12 & hour < 18 ~ "12:01 PM - 6:00 PM",
      hour >= 18 & hour <= 23 ~ "6:01 PM - 12:00 AM"
    ),
    delayed = if_else(dep_delay > 0, 1, 0)  # Using if_else for type consistency
  ) %>%
  filter(!is.na(dep_delay))  # Filter out rows with missing departure delay

# Calculating the proportions of delayed flights for each time interval
delay_proportions <- flights_delay_analysis %>%
  group_by(time_interval) %>%
  summarise(
    total_flights = n(),
    delayed_flights = sum(delayed),
    proportion_delayed = delayed_flights / total_flights
  ) %>%
  arrange(time_interval)  # Ensure intervals are in the correct order

# Print the results
print(delay_proportions)

Narrative: This plot shows the proportion of on-time and delayed flights for each airline, with blue bars for on-time flights and yellow bars for delayed ones.

Approach: I categorized flights into ontime (arrival delay zero or negative) and delayed (positive arrival delay). Then, I grouped the data by carrier and arrival status, calculated the proportions, and visualized the results in a stacked bar chart.

library(tidyverse)
library(nycflights13)

flights <- flights %>%
  mutate(arrival = ifelse(arr_delay <= 0, "ontime", "delayed"))

carrier_arrival <- flights %>%
  filter(!is.na(arrival)) %>% 
  group_by(carrier, arrival) %>%
  summarise(count = n()) %>%
  mutate(proportion = count / sum(count)) %>%
  ungroup()
## `summarise()` has grouped output by 'carrier'. You can override using the
## `.groups` argument.
ggplot(carrier_arrival, aes(x = carrier, y = proportion, fill = arrival)) +
  geom_bar(stat = "identity", position = "stack") +
  labs(title = "Proportion of On-Time and Delayed Flights by Carrier",
       x = "Carrier",
       y = "Proportion",
       fill = "Arrival Status") +
  scale_fill_manual(values = c("ontime" = "steelblue", "delayed" = "orange")) +
  theme_minimal()
